# FoundrySceneResizer


Foundry VTT scene resizer


```
  -s, --scene     Required. Scene name to resize.

  -w, --width     New Width.

  -h, --height    New Height.

  --scale         Factor to scale the scene. 2 for double, 0.5 for half, etc

  -f, --file      Path to scenes.db (should include file name). If not provided will use the root path and scenes.db
                  filename

  -o, --output    File name for the new scenes.db

  --help          Display this help screen.

  --version       Display version information.
```
